# 示例：一分钟职场荒诞喜剧

## 用户请求

```text
使用 $stephen-chow-skill，把“实习生在公司年会上冒充老板”写成一分钟短视频。
不要使用任何电影角色、台词或经典桥段。
```

## 1. 创作诊断

原始点子只有“错误身份”，还缺少角色为什么不能立刻澄清、事情如何升级，以及结局为何值得观众在意。

## 2. 原创设计

- **主角欲望**：实习生阿诚只想替同事争取被取消的餐补。
- **场景规则**：年会上，所有人必须使用公司管理黑话，谁说人话谁就“不够职业”。
- **主机制**：错误身份、地位反转。
- **辅助机制**：规则碰撞、道具回收。
- **关键道具**：一张写着“每人每天十二元”的餐补申请单。
- **情感落点**：真正的勇气不是继续扮演权威，而是用自己的身份说出真实问题。

## 3. 一分钟节拍表

| 时间 | 内容 | 功能 |
|---|---|---|
| 0-5秒 | 阿诚偷溜上台放申请单，聚光灯突然亮起，主持人高喊“新任CEO到了”。 | 视觉冲突与错误身份 |
| 5-15秒 | 阿诚想解释“我是实习生”，主持人将其解读为“从基层重新理解组织”。全场鼓掌。 | 建立黑话规则 |
| 15-28秒 | 高管请他宣布降本方案。他说“先把没用的会取消”，高管立刻删除了今晚年会流程。 | 第一次地位反转 |
| 28-42秒 | 高管追问裁员名单。阿诚举起餐补单说“先救每天十二块的”。大屏误投出“十二名核心人员”。 | 道具升级与误读 |
| 42-52秒 | 真老板在台下认真记笔记：“这个年轻人敢动核心层。”高管开始互相递辞职信。 | 最高升级 |
| 52-60秒 | 阿诚关掉话筒，用人话承认身份并说明餐补。沉默后，保洁阿姨第一个鼓掌。真老板划掉笔记，只留下“十二元”。 | 身份揭示与情感回收 |

## 4. 关键对白示例

```text
主持人：请您用一句话概括新的管理哲学。

阿诚：能不开的会，就别开了。

主持人（含泪）：去会议化管理。

阿诚：不是，我只是想说饭钱……

财务总监：明白，现金流优先于语言流。
```

## 5. 表演提示

- 阿诚越想说清楚，其他人越郑重，不要让他主动耍宝。
- 高管的笑点来自绝对认真，而不是夸张怪叫。
- 真老板记笔记时保持平静，让观众先于角色意识到风险。
- 结尾短暂停顿，给“十二元”从笑点转为真实诉求的空间。

## 6. 原创性说明

该方案使用错误身份、地位反转和规则碰撞等通用喜剧机制。人物、公司规则、道具、升级链、对白和结局均为独立设计，不依赖任何已有电影角色或场景才能成立。
